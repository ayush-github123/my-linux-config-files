
# What is this Repository

This repository stores all my Linux configuration files for different applications, software, and required `.conf` files as a backup.

These configurations are either:

* Downloaded from the internet and then customized by me, or
* Fully created and customized by me.

---

# Config Files in the Repo

```bash
1. L2M - Audio stream to mobile
2. boot-animation - Raw videos (.mp4/.webm)
3. keybindings - keybindings/default.conf
4. nvim
5. plymouth - Plymouth configuration
6. wallpapers
```
